Core interface for each ORM object to interface with CherryPy

class metadata.rest.orm.CherryPyAPI(*args, **kwargs)[source]¶

Core CherryPy interface for all orm objects

DELETE(**kwargs)[source]¶

Implements the DELETE HTTP method. Gets a single object based on kwargs, sets the deleted flag and saves the object.

GET(**kwargs)[source]¶

Implements the GET HTTP method. Returns the json object based on fields passed into kwargs.

POST(**kwargs)[source]¶

Implements the POST HTTP method. Gets the object similar to GET() and uses the request body to update the object and saves it.

PUT()[source]¶

Implements the PUT HTTP method. Creates an object based on the request body.
