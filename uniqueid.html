UniqueID Server.

The Call: curl ‘http://localhost:8000/getid?range=10&mode=file’

The Response: {“endIndex”: 9, “startIndex”: 0}

Select a different mode to get different unique IDs.

curl ‘http://localhost:8000/getid?range=10&mode=file’ curl ‘http://localhost:8000/getid?range=10&mode=upload’

The Response:

{“endIndex”: 9, “startIndex”: 0} {“endIndex”: 9, “startIndex”: 0}
